# 🎓 رادار الرصد الذكي - Student Radar

نظام ذكي لرصد طلبات الطلاب من قنوات ومجموعات Telegram.

## ✨ الميزات

- 🔍 رصد ذكي للطلبات والاستفسارات
- 🚫 فلترة تلقائية للإعلانات والروابط
- 📊 نظام نقاط متقدم
- 💬 دعم حسابات متعددة
- ☁️ يعمل على Render 24/7

## 🚀 التشغيل على Render

1. ارفع الكود على GitHub
2. أنشئ Web Service جديد على Render
3. أضف متغيرات البيئة:
   - `API_ID`
   - `API_HASH`
   - `TARGET_CHANNEL`
   - `SESSION_1`
   - `PORT=10000`
